## How It Works

1. **Enter your age** ‚Äî the app verifies you meet the minimum age requirement
2. **Choose your role** ‚Äî tell the app whether you are the younger or older partner
3. **Get your result** ‚Äî your compatible age range is displayed instantly in a clean, bold format

## Build From Source

If you'd like to build the app yourself:

```bash
git clone https://github.com/Akki-Maharaj/Heart-Formula.git
cd Heart-Formula
flutter pub get
dart run flutter_launcher_icons
flutter run
```

To build a release APK:
```bash
flutter build apk --release --target-platform android-arm64 --split-per-abi
```
The APK will be located at:
```
build/app/outputs/flutter-apk/app-arm64-v8a-release.apk
```
